# HisouSangokushiZero1_3
## 概要
悲愴三国志Zero バージョン3の開発プロジェクト  
開発技術：typescript,html
## ゲーム内容
悲愴感をコンセプトにしている制作中ゲームの悲愴三国志が完成するか不安なので作った簡易版、悲愴三国志Zeroの３つ目のバージョン  
三国鼎立時点からの三国志の魏・呉・蜀漢に加え燕・士氏、および周辺異民族でプレイ可能

隣接敵国エリアをクリックで侵攻、自国エリアをクリックで防衛にバフがかかる防衛専念  
人物の移動はマウスでドラッグ  
簡素でさっくり遊べるけど、戦略性はしっかり  
シナリオは１つ、勢力は１３あります  
人材層が後々まで厚く濃い魏、人材層は細々ながら前線管理しやすい呉、スタート時点は人材層が厚いがどんどん細る蜀漢、周辺民族でリードしている鮮卑、など  
登場人物が少なくパラメータもランクという一つの指標しかありませんが、陣営による特徴はそれぞれしっかりあります  
275年のゲームエンドまでに統一を目指そう
## 攻略テクニック
前線管理と侵攻人物の維持が重要  
まずは前線となる拠点数を増やさないように、領土の位置を整えるのが吉  
前線を突破されないように、前線に人物を配置  
侵攻人物は中枢所属の一番ランクが高い人物が担当するので、できれば主力を一人残すとやりやすい  
侵攻先を選ぶ際も注意、他国の防衛人物は１ターン前のが表示されている、侵攻実行時には変えられている  
CPUは基本、毎ターン防衛担当を変える、逆にこれを予測に利用できる  
防衛専念を上手く生かそう、クリック連打が最適手じゃない  
防衛専念では攻めてきた敵の人物が戦死しやすい、人物の層が戦況に直結するのでこれは重要  
逆に、敵の防衛専念に突っ込まないように、敵の休み状態を戦略に生かそう  
  
CPUチートなし、プレイヤーと同条件で、プレイヤーの入力の前に行動を確定させている  
CPUも前線の軍圧を感じ取り人物を配置する、でも出兵先の選定はテキトー
