# HisouSangokushiZero1_3
## 概要
悲愴三国志Zero バージョン3の開発プロジェクト  
開発技術：typescript,html
## ゲーム内容
悲愴感をコンセプトにしている制作中ゲームの悲愴三国志が完成するか不安なので作った簡易版、悲愴三国志Zeroの３つ目のバージョン  
三国鼎立時点からの三国志の魏・呉・蜀漢に加え周辺異民族でプレイ可能  
↓ゲームページを開くと最初はこういう見た目  
<img width="600" src="https://github.com/user-attachments/assets/6b4929e1-c752-443e-ae73-62aab7f75022" />  
↓勢力を選んでゲーム開始、ターンを進めるとこういう感じになります、蜀漢を選んだケース  
<img width="600" src="https://github.com/user-attachments/assets/c677c69d-c6d1-439f-be50-293eecc812e5" />  
隣接敵国エリアをクリックで侵攻、自国エリアをクリックで防衛にバフがかかる防衛専念  
人物の移動はマウスでドラッグ  
簡素でさっくり遊べるけど、戦略性はしっかり  
シナリオは１つ、勢力は１３あります  
人材層が後々まで厚く濃い魏、人材層は細々ながら前線管理しやすい呉、スタート時点は人材層が厚いがどんどん細る蜀漢、周辺民族でリードしている鮮卑、など  
登場人物が少なくパラメータもランクという一つの指標しかありませんが、陣営による特徴はそれぞれしっかりあります  
275年のゲームエンドまでに統一を目指そう
