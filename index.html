陣営中枢所属
ゲーム開始について
領土ボタンを押すと、その領土を有する国でプレイ開始になります
登場する全ての国でプレイ可能です
人物の配置について
陣営中枢所属から自陣営領土へ人物をドラッグで配置移動できます
逆に自陣営領土から陣営中枢所属へ配置移動もできます
既に人物が配置された自陣営領土へ人物を移動すると配置交換します
自陣営の人物のみ移動可能、配置変えに距離の制限はありません
ターン進行について
自領土か隣接領土のボタンをクリックでターンを消費します
人物の配置変更はターンを消費しません
自領土をクリックでそのターン防衛専念となり、他国から攻められた際に戦闘が有利になります
隣接する他国領土をクリックでそのターンその領土へ侵攻となり、勝利するとその領土を得ます
ただし休み状態であれば、ターン進行のためにクリックは要りますが、防衛専念も侵攻もしません
隣接しない他国領土はクリックすることができません
レイアウトモードについて
fixモードとfitモードがあり、デフォルトはfixモードです
fixモードはレイアウトが固定で、画面サイズが小さい場合は全体が縮小されます
fitモードはレイアウトが可変で、画面サイズにに合わせながら地図が大きく表示されるようになります
左上のボタンで切り替え、「▲」になっていればfixモード、「▼」になっていればfitモードです
初期化について
右上の初期化ボタンで盤面が初期化され、ゲーム開始前の状態に戻ります
セーブ機能がないことに留意してください
勝利・敗北について
勝利条件：プレイヤー勢力領土全制覇(制覇勝利) or 西暦275年春までプレイヤー勢力存続(存続勝利)
敗北条件：プレイヤー勢力領土全失陥
陣営の行動について
行動順：魏 → 呉 → 蜀漢 → 燕 → 士氏 → 南蛮 → 東夷 → 北匈奴 → 羌 → 氐 → 南匈奴 → 鮮卑 → 烏丸
※行動時に既に滅亡していた場合は行動実行できません

防衛・侵攻について
侵攻を受けた際の防衛は対象の領土に配置された人物があたります
侵攻は陣営中枢所属のうちランクが一番高い人物の中でより前に配置されている人物があたります
防衛・侵攻で候補となる人物が明示的にいなかった場合、ランク0の無名の人物が担当していると処理されます

休み状態（国力回復中状態）について
休み状態は出兵して国力を減らした状態を表現しています
「休み」の表示が出て、そのターン数行動ができません
攻撃もできませんが防衛専念状態にもなりません
